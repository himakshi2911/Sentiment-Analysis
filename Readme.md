# Sentiment Analysis Project

This project predicts whether a movie review is positive or negative using Machine Learning.

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Logistic Regression
- TF-IDF Vectorizer

## Files

- train.py
- test_model.py
- model.pkl
- vectorizer.pkl

## Example

Input:
This movie was amazing!

Output:
positive
